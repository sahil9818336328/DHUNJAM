## Installation

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sahil9818336328/DHUNJAM.git

2. **Navigate to project directory**
   ```bash
   cd DHUNJAM
   
3. **Install dependencies**
   ```bash
   npm install

4. **Start the server**      
   ```bash
   npm run dev
